提醒个点，使用注解的方式写切面时，增加了一个空方法，即：  
```java

    @Pointcut("execution(* org.bk.inventory.service.*.*(..))")
    public void serviceMethods(){

    }

```
在使用xml配置的话，就不需要这个方法了，serviceMethods方法名是后面配置切点的引用。

如果不想引入spring的话，可以直接使用aspectj或者jboss aop。
